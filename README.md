A minimal agent running in your terminal.

## Quick Start

```bash
uv tool install git+https://github.com/kowyo/mini-agent.git@main
export ANTHOPIC_API_KEY=<your_anthropic_api_key>
mini-agent
```

## Development

```bash
make prepare
export ANTHOPIC_API_KEY=<your_anthropic_api_key>
uv run mini-agent
```
